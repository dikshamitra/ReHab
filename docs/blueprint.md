# **App Name**: Recovery Circle

## Core Features:

- Goal Setting: Allow users to set personalized goals related to quitting their addiction (e.g., days without substance, reduced consumption).
- Progress Tracking: Visually track progress toward goals using charts and progress bars.
- Daily Affirmations: Offer daily inspirational quotes or affirmations to motivate users.
- Coping Strategy Suggestions: Use generative AI to suggest alternative coping strategies based on the user's addiction type, progress, and stated triggers. The generative AI tool will select from a list of known, established coping techniques.
- Anonymous Chat: Facilitate anonymous one-on-one chat between users and counselors (no database component).
- Resource Directory: Provide a list of helpline numbers and websites as resources.
- Gamified Rewards: Gamified system to reward successful completion of goals and milestones.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to evoke trust and serenity.
- Background color: Light gray (#F0F4F8) for a clean and unobtrusive backdrop.
- Accent color: Soft green (#81C784) to indicate progress and positive reinforcement.
- Body and headline font: 'Inter' sans-serif for a modern and readable experience.
- Use clean, simple icons to represent different features and actions.
- Prioritize a clean and well-organized layout with clear visual hierarchy for easy navigation.
- Incorporate subtle animations to provide feedback and enhance the user experience, such as progress bar updates or celebratory animations upon achieving milestones.
